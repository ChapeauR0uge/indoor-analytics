pathDistance

Computes distance between two paths by projecting positions of the second (acquired) path onto the first (reference) one. This allows to compare paths with different location counts.

Arguments

ArgumentsTypeDescription
referencePathFeature <LineString>the reference path
comparedPathFeature <LineString>the compared path

Returns

ErrorVector - a vector projection on the reference path
npm i --save @indoor-analytics/path-distance

Example

                  
                     import {pathDistance} from '@indoor-analytics/path-distance';
 
 const referencePath = "euratechPath1";
 const comparedPath = "euratechPath2";
 
 const vectors = pathDistance(referencePath, acquiredPath);

Algorithm Formalization

For each acquired path location, we create a projection on the reference path; first path location will always match first reference path, the same for the last location.

A projected position is obtained by:
  1. getting distance between the current path position and the path origin;
  2. computing the ratio position distance / path length;
  3. computing the equivalent distance regarding the reference path;
  4. projecting the position on the reference path.
