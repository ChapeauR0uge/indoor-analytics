pathDistance

Computes distance between two paths by projecting positions of the second (acquired) path onto the first (reference) one. This allows to compare paths with different location counts.

Arguments

ArgumentsTypeDescription
referencePathFeature <LineString>the reference path
comparedPathFeature <LineString>the compared path

Returns

ErrorVector - a vector projection on the reference path
